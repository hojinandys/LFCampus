# LFCampus - 찾아줘 캠퍼스!

## 프로젝트 소개

**찾아줘 캠퍼스!**는 대학생을 위한 캠퍼스 분실물·습득물 매칭 웹사이트입니다.

교내에서 물건을 잃어버렸을 때 학생지원팀, 도서관 안내데스크, 건물 경비실 등 여러 장소를 직접 확인해야 하는 불편함을 줄이기 위해 제작했습니다.
사용자는 분실물과 습득물 목록을 확인하고, 물건을 직접 등록하며, 보관 장소와 문의 방법을 확인할 수 있습니다.

본 프로젝트는 Bootstrap 기반의 FlexStart 템플릿을 활용하여 제작하였으며, 주제에 맞게 디자인과 내용을 수정했습니다.

---

## 주요 기능

### 1. 메인 페이지

* 캠퍼스 분실물 매칭 서비스 소개
* 분실물 찾기 버튼 제공
* 서비스 목적과 주요 기능 안내

### 2. 최근 분실물·습득물 목록

* 최근 등록된 분실물과 습득물을 카드 형태로 표시
* 분실물과 습득물을 배지 색상으로 구분
* 물건명, 카테고리, 장소, 날짜, 특징 정보 제공
* 채팅 문의하기 버튼 제공

### 3. 분실물·습득물 등록

* 분실물 등록 폼 제공
* 습득물 등록 폼 제공
* 물건명, 카테고리, 장소, 날짜, 특징, 사진 첨부 입력 가능
* 현재는 화면 구현 중심이며 실제 데이터 저장 기능은 구현하지 않음

### 4. 채팅 팝업 UI

* 채팅 문의하기 버튼 클릭 시 화면 위에 채팅창 팝업 표시
* 물건 정보, 채팅 입력창, 안전 거래 수칙 영역으로 구성
* 실제 메시지 저장 및 전송 기능은 추후 서버와 데이터베이스 연결을 통해 확장 가능

### 5. 보관 장소 안내

* 학생지원팀
* 도서관 안내데스크
* 건물 경비실

### 6. FAQ 및 문의

* 분실물 수령 방법
* 습득물 보관 장소
* 채팅 기능 안내
* 문의 양식 제공

## 코드 구현 설명

### 최근 분실물·습득물 목록

최근 분실물·습득물 목록은 Bootstrap의 `row`, `col-lg-4`, `col-md-6` 클래스를 사용하여 반응형 카드 형태로 제작했습니다.
큰 화면에서는 한 줄에 3개의 카드가 보이고, 태블릿에서는 2개, 모바일에서는 1개씩 보이도록 구성했습니다.

각 카드에는 다음 정보가 들어갑니다.

* 분실물 또는 습득물 구분 배지
* 물건 아이콘
* 물건명
* 카테고리
* 발견 또는 분실 장소
* 날짜
* 특징
* 채팅 문의하기 버튼

분실물은 주황색 배지, 습득물은 초록색 배지로 구분하여 사용자가 쉽게 알아볼 수 있도록 했습니다.

---

### 분실물·습득물 등록 폼

등록 영역은 분실물 등록 폼과 습득물 등록 폼으로 나누어 제작했습니다.

분실물 등록 폼에는 다음 입력 요소를 사용했습니다.

* `input type="text"`: 물건명, 분실 장소 입력
* `select`: 카테고리 선택
* `input type="date"`: 날짜 선택
* `textarea`: 물건 특징 입력
* `input type="file"`: 사진 첨부
* `button type="submit"`: 등록 버튼

습득물 등록 폼은 발견 장소와 보관 장소를 추가로 입력할 수 있도록 구성했습니다.

폼 제출 시에는 JavaScript의 `submitDemoForm()` 함수를 사용하여 페이지 새로고침을 막고 등록 완료 알림을 표시하도록 했습니다.

---

### 채팅 팝업 UI

채팅창은 별도 페이지가 아니라 메인 페이지 위에 뜨는 팝업 형태로 만들었습니다.

채팅창은 기본적으로 `display: none` 상태로 숨겨져 있다가, 사용자가 `채팅 문의하기` 버튼을 누르면 JavaScript를 통해 `is-open` 클래스가 추가되어 화면에 나타납니다.

채팅창은 다음 3개의 영역으로 구성했습니다.

1. 물건 정보 카드
2. 채팅 입력창
3. 안전 거래 수칙

현재 채팅창은 디자인과 사용자 흐름을 보여주기 위한 UI이며, 실제 메시지 저장 및 전송 기능은 구현하지 않았습니다.

---

### 카테고리 목록

카테고리 목록은 `select`와 `option` 태그를 사용하여 만들었습니다.

사용한 주요 카테고리는 다음과 같습니다.

* 학생증
* 지갑
* 전자기기
* 우산
* 책
* 기타

검색 영역에서는 사용자가 원하는 카테고리를 선택할 수 있도록 했고, 등록 폼에서는 물건 등록 시 카테고리를 필수로 선택하도록 `required` 속성을 사용했습니다.

## 실행 방법

1. 프로젝트 파일을 다운로드합니다.
2. `index.html` 파일을 브라우저에서 실행합니다.
3. 상단 메뉴를 통해 각 섹션 또는 추가 페이지로 이동할 수 있습니다.

또는 GitHub Pages를 사용하면 웹사이트 주소로 접속할 수 있습니다.
